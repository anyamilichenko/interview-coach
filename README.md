# Multi-Agent Interview Coach

Система для проведения технических интервью с использованием нескольких AI-агентов.

## Архитектура

Система построена на основе трех специализированных агентов с четким разделением обязанностей:

### Агент-Интервьюер (Interviewer)
- Ведущий интервью, задает технические вопросы
- Адаптирует сложность вопросов на основе ответов кандидата
- Обрабатывает off-topic и возвращает к теме интервью
- Отвечает на встречные вопросы кандидата

### Агент-Наблюдатель (Observer)
- Анализирует каждый ответ кандидата "в фоне"
- Обнаруживает галлюцинации и технические ошибки
- Оценивает уровень уверенности кандидата
- Выявляет пробелы в знаниях
- Дает рекомендации интервьюеру для следующих вопросов

### Агент-Оценщик (Evaluator)
- Генерирует структурированный фидбэк по итогам интервью
- Формирует вердикт (уровень, рекомендация по найму)
- Анализирует hard и soft skills
- Создает персонализированный roadmap для развития

## Требования

- Python 3.8+
- Mistral API ключ (можно получить на [console.mistral.ai](https://console.mistral.ai))
- 2+ ГБ свободной памяти
- Интернет-соединение для работы с API

## Использование

Мы можем использовать 3 режима:

1. Интерактивный режим

Запускаем:
```bash
python run_interview.py
```
Выбираем режим 1 и следуем инструкциям.

2. Режим сценария (из файла JSON)

Создайте JSON файл со сценарием (например test_scenario.json):
```json
{
  "participant_name": "Иван Иванов",
  "position": "Python Developer",
  "grade": "Middle",
  "experience": "3 года опыта",
  "user_responses": ["ответ1", "ответ2", "..."]
}
```
Запустите:
```bash
python run_interview.py
```
Далее выберите режим 2 и укажите путь к файлу.

3. Запуск тестового сценария

Запустите:
```bash
python run_interview.py
```
А далее он все сделает сам.
