# Multi-Agent Interview Coach

Система для проведения технических интервью с использованием нескольких AI-агентов.

## Архитектура

Система состоит из трех агентов:
1. **Interviewer** - ведет диалог, задает вопросы
2. **Observer** - анализирует ответы кандидата, проверяет факты
3. **Evaluator** - генерирует финальный фидбэк

## Требования

- Python 3.8+
- Mistral API ключ
